# 프로젝트 내용
### 모두 가볍고 즐겁게 사용자들이 서로의 의견을 주고받을 수 있고 Enjoy&Easy(E&E) 설문을 등록하여 다른 사용자들이 참가하여 즐길수 있는 사이트입니다.

<br/>

# 구현 기능

<br/>

## E&E Survey
- 등록 (O)
- 검색 (O)
- 추천
- 참여 (O)
- 디테일 조회

<br/>

## 게시판
- 등록
- 검색
- 댓글
- pagenation
- 상세

<br/>

## 문의
- 1대1 문의
- Q&A 문의

<br/>

## 관리자
- 유저(사용자 계정) 관리
- 문의 관리
- 게시글 관리
- 댓글 관리
- 공지사항 등록
- E&E survey 관리
